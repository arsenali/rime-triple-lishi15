# rime-triple-lishi
李氏三拼方案，三码输入一个带调音节

调整为26键和15键两个版本，电脑上主要用26键版，手机主要用15键版。

两个版本都用三码输入一个带调音节，26键版为“声+韵+调”方案，第三码声调可省略；15键版为“声+介调+韵”方案，声调不可省略。26键用于qwerty键盘，15键可用左手、右手、双手操作，可用于数字键盘和qwerty键盘。

详见阿森纳里的简书“李氏三拼”专题。
